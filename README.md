# RUSit - Simple Russian keyboard app

## Requirements

- Java JRE 1.8 (I use Lambda expressions)

## Run

- Method 1: Open project in IDE (e.g. Eclipse) and click Run As Java App.
- Method 2: Download JAR file and run it
